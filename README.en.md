# Kecomu • AI for Web Development

## Description
**Kecomu** is a web platform designed to assist developers in creating HTML, CSS, and JavaScript code through artificial intelligence. Currently, the project is in the visual interface development phase, featuring a complete customization and accessibility system.

----------------------------------------------

## Version
<img src="https://img.shields.io/badge/Version-1.0.0.0--Alpha-2E0008?style=for-the-badge&logo=github&labelColor=000000&color=FFC8C8" />

---

## Purpose

**Kecomu** was born with the idea of simplifying web development through three specialized versions:
  
  - Nova • The "Nova" version of Kecomu is specialized in creating flexible and optimized code!
  - Kero • The "Kero" version of Kecomu is specialized in creating creative code with attractive graphic design!
  - Umwarimu • The "Umwarimu" version of Kecomu is a guided learning AI, allowing you to create while you learn!

---

## Current Features (Visual Interface)

- Modern design with sleek aesthetics and smooth animations
- Light/dark mode with cookie persistence
- Three animation quality levels: low, medium, and high
- Visual clarity mode: improves accessibility by removing blur effects, increasing contrast and text size
- Responsive design adaptable to mobile devices and tablets
- Preference system synchronized across all pages
- News module with update history
